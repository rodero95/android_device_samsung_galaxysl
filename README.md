android_device_samsung_galaxysl

Development for Samsung Galaxy SL (I9003)

	·cm-10.1: to build CyanogenMod 10.1 with the 3.x kernel.

	·phablet-10.1: to build the CyanogenMod 10.1 base for Ubuntu touch.
